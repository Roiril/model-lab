# 1209 レーザーカット手順 — かわいいロボット

このフォルダ（`kit-1209`）を **USB かSlackかメールでレーザーPCに渡して**、上から順にやるだけ。

## このフォルダの中身
| ファイル | 用途 |
|---|---|
| **laser-bot.svg** | ← Ruby に読み込む本番カットデータ（これがメイン） |
| laser-bot.pdf | SVGがうまく読めない時の予備（1:1 ベクター） |
| preview.png | 何を切るかの確認用（顔つき展開図） |
| 手順.md / 手順.html | これ |

切り出すのは **6枚のパネル**（前・後・左・右・上・下）。組むと **68×56×56mm の箱ロボ**（立方体よりちょっと横長）。**6面すべてに違う表情**を彫刻（画像から輪郭抽出したかわいい顔）。

> **顔の見え方**：顔は彫刻（フロスト）なので、**濃色/カラーのアクリル**だと彫った所が白っぽく光って表情がくっきり出る（透明・白だとコントラストが弱く見えにくいので、黒・紺・濃緑などの濃色がおすすめ）。プレビューは黒＝彫刻部分の表示で、実物は地が濃色・顔が白っぽくなる。

---

## 準備するもの
- [ ] **3mmアクリル板**（最低 220×130mm 入る大きさ。**濃色（黒・紺・濃緑など）**だと顔が一番映える）
- [ ] アクリル接着剤（あれば。無くてもフィンガーで仮組みは効く）

---

## 手順（上から順に）

### A. 現地セット
1. [ ] **窓を開ける**（ダクト排気）
2. [ ] レーザーカッターの **電源 ON**
3. [ ] アクリル板を加工エリアに **置く**
4. [ ] 上下ボタンで **ワークエリアを上げる**
5. [ ] **ピント調整器具**をヘッドに引っかけ、板を少しずつ上げる → 器具が外れて倒れた高さが焦点。器具を外す
6. [ ] 上下左右キーで **ヘッドを切りたい位置へ** 移動

### B. Ruby（ソフト）
7. [ ] Ruby を開く（`share.fms` アカウント。2段階認証が出たら現地PCのChromeのGmailで番号確認）
8. [ ] **laser-bot.svg を管理画面にドラッグ&ドロップ**（読めなければ laser-bot.pdf で）
9. [ ] 寸法確認：パネル1枚の横が **約60mm** で出ているか。違ったらスケール調整
10. [ ] **色で加工を割り当てる**（下表）
11. [ ] **ジョブ画面**で図面をヘッド位置［＋］に合わせ、**キュー作成**
12. [ ] **再生ボタンで加工開始**

#### 色の割り当て（重要）
| 色 | 加工 | 設定 |
|---|---|---|
| **赤（細線）** | **カット** | 3mmアクリル用のパワー/スピード |
| **黒（顔）** | **彫刻（engrave）** | 既定の彫刻設定 |

> 彫刻するなら **アクリルの保護シートは剥がす**（剥がさないと顔がきれいに出ない）。

### C. 組み立て
13. [ ] 6枚を箱状に組む（フィンガーをはめ込む）。前面（顔）が正面
14. [ ] きつければ無理に押さない／緩ければ接着剤で固定

---

## ⚠ 安全（必ず守る）
- **加工中は絶対に離席しない**。アクリルは発火することがある
- 火が出たら 10秒程度は様子見 → 続くなら **1209のエアロゾル消火器**。広がったら廊下の消火器
- 加工後のパーツは熱い・エッジが鋭いことがある

---

## うまくフィットしなかった時（次回用メモ）
- フィンガーが **きつすぎ** → `lib/laser_core.py` の `KERF` を **上げる**（例 0.1→0.15）
- **緩すぎ** → `KERF` を **下げる**（例 0.1→0.05）
- 直したら `python models/laser-bot/make_kit.py` で再生成
