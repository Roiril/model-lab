# グローバルルール（他エージェント向け）

Claude Code は `~/.claude/CLAUDE.md` を参照するためこのファイルは主に Cline / Roo Code 用。矛盾する場合はグローバル CLAUDE.md が優先される。

1. **計画の永続化**: 実装計画やタスクリストは `.agent/plans/` および `.agent/tasks/` に実ファイル（`YYYY-MM-DD_<slug>.md`）として作成・更新すること
2. **客観性の維持**: コード実行結果やエラーの評価は、コンソール出力 / リンター出力 / API レスポンスなど客観的データに基づき判断すること
3. **出力制限**: 明示的な指示がない限り、コード全体をチャットに出力せず、変更箇所のみを簡潔に提示すること
4. **Windows 環境のエンコーディング**:
    - **PowerShell (.ps1)**: 日本語を含む場合は UTF-8 with BOM で保存
    - **バッチ (.bat)**: 言語環境依存の文字化けを防ぐため ASCII のみで作成。メッセージ表示は PowerShell に委譲
5. **PowerShell 配列コアーション**: 0件または1件になり得るコマンドレット結果は `@(...)` で配列型を強制すること
